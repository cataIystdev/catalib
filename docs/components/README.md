# Компоненты catalib

Документация по слоям. Структура зеркалит `src/catalib/`. В каждом документе
есть раздел «Связи».

- [manifest](manifest.md) — манифест и валидация метаданных
- [bundler](bundler.md) — обход исходников и компиляция в один файл
- [runtime](runtime.md) — встраиваемый загрузчик
- [support](support.md) — мини-фреймворк, встраиваемый в плагин
- [deploy](deploy.md) — доставка на устройство через dev server
- [cli](cli.md) — команды `build`, `watch`, `init`
- [scaffold](scaffold.md) — генерация шаблона проекта

Рабочие сценарии — в [guides](../guides/).
