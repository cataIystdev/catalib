# Компонент: runtime

Назначение: встраиваемый загрузчик, исходник которого целиком копируется в
собранный плагин и исполняется под Chaquopy на устройстве.

## Состав

- `bootstrap.py` — `_CatalibFinder`/`_CatalibLoader` (поверх `importlib`),
  функция `catalib_install`: восстановление идентичности модуля (литерал
  `__name__` метаданных перетирает имя модуля), регистрация finder в
  `sys.meta_path`, исполнение тела корневого модуля, импорт точки входа,
  поднятие класса плагина на верхний уровень, регистрация в `linecache`.
- `__init__.py` — `get_bootstrap_source()` для компилятора.

Зависит только от стандартной библиотеки; подпроцессы не используются
(в Chaquopy недоступны). Подтверждено на устройстве (T-011).

## Связи

- Зависит от: стандартная библиотека (`importlib`, `linecache`, `sys`),
  опционально SDK `base_plugin` (определение класса плагина).
- Используется: `bundler.compiler` (встраивает исходник).
- Связанные документы:
  [ADR-0002](../architecture/decisions/ADR-0002-bundler-meta-path.md),
  [эмпирические данные](../architecture/evidence/T-011-probe-result.json),
  память: ограничения Chaquopy.
