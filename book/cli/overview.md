# Обзор CLI

После установки доступна команда `catalib`:

```bash
catalib --help
```

| Команда | Назначение |
|---------|------------|
| [`catalib build`](build.md) | собрать модульный плагин в один файл |
| [`catalib watch`](watch.md) | пересобирать при изменениях, опционально деплоить |
| [`catalib init`](init.md) | создать шаблон модульного плагина |
| `catalib version` | напечатать версию catalib |

Запуск без аргументов показывает справку.

## Коды возврата

| Код | Значение |
|-----|----------|
| `0` | успех |
| `1` | ошибка сборки/инициализации/деплоя (сообщение в stderr) |
| `2` | ошибка разбора аргументов (показана справка) |

Это позволяет использовать catalib в скриптах и CI:

```bash
catalib build --project . || exit 1
```

## Сообщения

Информационные сообщения идут в stdout, ошибки — в stderr с пометкой
(`Ошибка сборки: ...`). Это удобно для парсинга в автоматизации.

## Запуск как модуль

Эквивалентно консольной команде:

```bash
python -m catalib --help
```
